

CSS
(Cascading Style Sheet)

HTML 문서에서 style 속성을  추가하여 시각적으로 꾸며주는 역할을 한다

------------------------------------------------------------------------

CSS문법
    선택자(내가 어떤 요소에 스타일 속성을 줄 것인지)
    ...CSS 문법
    속성명: 속성값
    color: "red"

ex>
p {
    text-align: center
}
-> 해당 웹 문서의 모든 p 태그의 텍스트를 가운데 정렬해라

------------------------------------------------------------------------

CSS 적용 방법
    1. 내부 스타일링
        > 하나의 html 파일 안에 같이 작성하는 것
        > head 태그 안에 style 태그를 통해 CSS 문법 적용
        ex> <html>
                    <head>
                        <style>
                            ...CSS 문법 작성
                        </style>
                    </head>
                </html>

        >>> 내부 스타일링은 코드가 길고 복잡해지고 찾기 힘들기 때문에 정말 간단한 게 아닌 이상 거의 사용되지 않는다


    2. 외부 스타일링
        >html 외부에 파일을 생성하는 것 (이름은 반드시 .css 로 끝나야함)
        >link 태그를 사용하여 사용하고자하는 html에 연결하여 사용할 수 있다
        ex> <head>
                    <link href="경로" rel="stylesheet" />
                </head>

        >>> 외부 스타일링이 "가장 많이 쓰는 CSS적용 방법"
        이유는 : 파일을 새로 만드는 것이기에 다른 html 에도 적용해서 재사용이 가능하다/ 분리해서 독립적으로도 이용이 가능하다
        단점은 - css파일이 많아져서 관리해야 하는 파일이 많아질 수도 있다 >> 단점을 극복하기 위해서 이름 명시를 잘해야한다


    3. 인라인 스타일링
        >태그(요소) 안에 직접 style 속성을 정의하여 사용
        >*가장 우선 순위가 높습니다 -> 그렇기에 다른 애들의 속성을 다 무시해버릴 수 있다
        (제일 세다고 볼 수 있다 - 직접적으로 속성을 부여하는 것이기 때문)
        ex> <p style="text-align: left; color: blue;"></p>

        >>> 가장 사용되지 않는 적용 방법이다, 코드가 더러워지고 가독성이 굉장히 좋지 않다
        인라인의 장점은 왔다갔다 안 하고 위에서 순서대로 코드를 읽을 수 있다
