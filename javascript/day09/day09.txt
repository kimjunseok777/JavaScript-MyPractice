
캡쳐링과 버블링  ** 취업 면접 단골 질문
    이벤트 전파 :
        ( "자식"을 클릭했는데 부모가 클릭  -->  버블링 )
        ( "부모"를 클릭했는데 자식이 클릭  -->  캡쳐링 )

    버블링 :
        "자식" 요소를 클랙했을 때 "부모" 요소로 이벤트가 전파되는 것
        버블링은 제어할 수 있다(이벤트 전파 방지) :
        event.stopPropagation()
        ==> 버블링은 자주 일어나니 알아두자

    캡쳐링 :
        ( 기본값이 false로 되있기에 크게 신경쓸 필요는 없다 )
        "부모" 요소를 클릭했을 때 "자식" 요소로 이벤트가 전파되는 것
        addEventListener(type, func, capture( true or false )) --> 이렇게 작성하면 되는데, 기본값이 false이다

        캡쳐링은 기본 값이 false 라서 꺼져있다, 하지만
        만약에 부모요소를 클릭했을 때 자식요소로 이벤트를 전파해야 되는 요직이 필요하다면
        capture 옵션을 true로 해주면 된다
        --> (사실상 이렇게 할 일 거의 없다고 보면 된다)
